# Array

## Shift elements

- 第一層：基本功
    - 核心工具：k % n（處理大 $k$ 值）和 Slicing (切片)。
    - 方向互換：
        - 左移 $k$ 位：等同於 右移 $n - k$ 位。
        - 左移語法：arr[k:] + arr[:k]
        - 右移語法：arr[-k:] + arr[:-k]
        - Space $O(n)$
- 第二層： In-Place Shift
    * 三步翻轉法： $O(n)$ 時間、 $O(1)$ 空間。以右移 $k=3$ 為例，數組為 [1, 2, 3, 4, 5, 6, 7]：
        1. 翻轉整個數組：[7, 6, 5, 4, 3, 2, 1]
        2. 翻轉前 $k$ 個：[5, 6, 7, 4, 3, 2, 1]
        3. 翻轉剩下的：[5, 6, 7, 1, 2, 3, 4]
    * 如果是左移？ 順序微調即可（先翻轉前 $k$ 個，再翻轉後 $n-k$ 個，最後全翻轉）。
